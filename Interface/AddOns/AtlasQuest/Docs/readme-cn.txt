关于 AtlasQuest:
=================

AtlasQuest 是一个基于 Atlas 的扩展插件。它能够显示每个副本，战场和户外首领的相关任务，
并且有注释能够告诉你如何完成这个任务。而副本背景介绍和简单的boss攻略能让你的副本更轻松。

AtlasQuest的原作者是 Asurn，现在由 Thandren（亦称 Mystery8）维护更新。

AtlasQuest 中的第一手游戏数据从 WoWhead.com 和 WoWpedia.org 活得。
