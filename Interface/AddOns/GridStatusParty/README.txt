Description
-----------

GridStatusParty adds statuses to Grid for party roles:

* Party Leader (light blue box or "PL" text)
* Raid Leader (tan box or "RL" text)
* Raid Assistant (yellow box or "RA" text)
* Master Looter (white box or "ML" text)

Use the Grid options menu ("/grid") to assign the statuses to indicators, or to
change colors, text, or other settings.


Dependencies
------------

GridStatusHealingReduced is a plugin for Grid, which can be downloaded
from either of these pages:

* http://www.wowinterface.com/downloads/info5747-Grid.html
